# Apk Download

[https://fir.im/r26f](https://fir.im/r26f)

# Completed 

* 扫码登录功能
* 获取文章等信息并展示功能
* Markdown编辑器以及预览功能
* 发表文章和评论，以及点赞等功能
* 对Fragment的内存恢复做了处理
* 图片压缩上传
